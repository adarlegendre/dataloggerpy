FIX FOR DAILY EMAILS AT MIDNIGHT:
----------------------------------

Option 1: Via Django Admin or Web Interface
1. Go to Configuration -> Notification Settings
2. Set "Days of the Week" to BLANK (or delete the content)
3. Set "Notification Times" to: 00:00
4. Save

Option 2: Via Command Line
python manage.py shell
>>> from app.models import NotificationSettings
>>> s = NotificationSettings.objects.first()
>>> s.days_of_week = ''  # Empty = all days
>>> s.notification_times = '00:00'  # Midnight
>>> s.save()
>>> exit()
